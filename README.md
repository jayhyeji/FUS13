# CLI 명령어

* pwd: 현재 디렉토리 확인하기
* cd: 디렉토리 이동
* mkdir: 디렉토리 생성하기
``` js
mkdir fus13(폴더명)
```
* touch : 파일 생성
* echo: 파일 수정
* mv : 이동하기
* cp: copy의 줄임말 
* rmdir : 비어있는 디렉토리 지우기
* rm : 제거할 파일/ 폴더 
* ls: List Segments
* cat: 파일 내용을 확인하기
